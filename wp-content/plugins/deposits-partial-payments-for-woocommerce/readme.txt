== Description ==
Deposits & Partial Payments for WooCommerce helps customers to pay for products using a deposit or a partial payment.


== HOW TO  manage WooCommerce Deposits settings? ==
After activating Deposits & Partial Payments for WooCommerce plugin, go to the Deposits menu in WordPress left menu bar. Here you can manage Deposits settings with the available options provided.

== BASIC FEATURES ==
🔹 Enable or disable deposits store wide with a single click
🔹 Allow customers to place order with deposits or full amount
🔹 Can control the use of deposits for unauthenticated users.
🔹 Deposits can be done by percentage or fixed amount
🔹 Custom deposit settings at products level as well as global setting
🔹 Enable or disable deposits for specific products
🔹 Customers can pay the balance amount later by logging in to their account.
🔹 Can disable specific payment gateways at checkout for deposit product orders
🔹 Can customize the plugin labels & texts
🔹 Translation ready

== INSTALLATION ==

Installing "Deposits & Partial Payments for WooCommerce" can be done either by searching for "Deposits & Partial Payments for WooCommerce" via the "Plugins > Add New" screen in your WordPress dashboard, or by using the following steps:

1. Download the plugin via WordPress.org
1. Upload the ZIP file through the 'Plugins > Add New > Upload' screen in your WordPress dashboard
1. Activate the plugin through the 'Plugins' menu in WordPress

== Frequently Asked Questions ==
= What type of deposits are available? =
You can collect deposits based on a fixed amount or a percentage of the product price
= Are deposits refundable?  =
Yes, but it will need to be refunded manually from the order.
